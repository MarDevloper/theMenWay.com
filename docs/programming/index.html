تعرف على أشهر لغة برمجة في مجال الويب ومجالات استخدامها.

بلا شك أنك سمعت أكثر من مرة عن لغة جافا سكريبت - JavaScript وقد تتساءل ماهي بالضبط؟

حسنا لغة جافا سكريبت - JavaScript هي من بين أكثر لغات البرمجة شهرة على مستوى العالم لعام 2022 وهي إحدى أهم لغات الويب الأساسية الى جانب CSS و HTML والتي تستخدم في بناء صفحات الويب في كافة المواقع الإلكترونية.

HTML: HyperText Markup Language

اذا أردت بناء صفحة ويب فإن أول شيء عليك استخدامه هي لغة HTML وهي لغة تعتمد على الحافظات أو tag وتمكنك من بناء وتنسيق الهيكل الأساسي لصفحة الويب والذي يتكون أساسا من العناوين الكبيرة، الصور والنصوص.

لغة تنسيق المحتوى CSS :

الآن بعد قمت ببناء الهيكل الأساسي لصفحة الويب يأتي دور التنسيق. فاستخدام مجموعة أوامر مكتوبة بلغة CSS - وهي سهلة للغاية ويمكنك تعلمها في وقت قياسي - تستطيع تنسيق النصوص والعناوين من خلال تغيير لون الخط وشكله وحجمه، كما يمكنك تغيير الخلفية مثلا جعلها في لون واحد أو عدة ألوان متدرجة أو ربما تجعل صورة معينة كخلفية للنص الذي تود أن يظهر للمستخدم. أكثر من ذالك تستطيع إضفاء التأثيرات على أي مكون من مكونات صفحة الويب التي أنت في صدد بنائها.

لغة البرمجة جافا سكريبت - JavaScript

جافا سكريبت تعتبر مكون أساسي داخل كل متصفح ويب، سواء كنت تستخدم متصفح Chrome أو Firefox أو Safari فإن جميع هذه المتصفحات تدعم لغة جافا سكريبت. تم تطوير JavaScript من قبل شركة نيتسكاب - Netscape الأمريكية عام 1995. تستخدم أساسا في جعل صفحات الويب أكثر دينامية على سبيل تحميل النصوص والصور عند الضغط على زر محدد على صفحة الويب عن طريق استخدام Ajax.

اذا كنت تسمع لأول هذه الكلمة فهي اختصار ل - Asynchronous JavaScript and XML - والتي تمكنك من جلب البيانات من السيرفر - على سبيل المثال النصوص والصور - دون أن يؤدي ذالك الى تعطيل مؤقت لتحميل صفحة الويب. ما يعني أن المستخدم بإمكانه تصفح المحتوى الأساسي ليتما يتم جلب باقي المحتوى من السيرفر.

ماهي مميزات لغة الجافا سكريبت؟
  • لغة تفسيرية عالية المستوى اي أن متصفح الويب يقوم بتنفيذ الأوامر المكتوبة بها بشكل تتابعي على عكس لغات أخرى ك C++ و Java.
  • تعمل في جهة العميل الذي يكون عادة هو متصفح الويب على خلاف لغات أخرى تعمل حصرا في جهة الخادم من قبيل PHP و Python.
  • تدعم البرمجة كائنية التوجه بحيث تسهل على المطورين بناء برامج معقدة مع سهولة الصيانة.
أهم استخدامات لغة JavaScript :
  • بناء صفحات ويب تفاعلية مع ميزة تحميل المحتوى عند الطلب.
  • تطوير الألعاب والتطبيقات التي تستهدف منصات الويب ونعني بذالك متصفحات الويب الشهيرة ك Google Chrome و Firefox.
  • بناء تطبيقات الاندرويد باستخدام مكتبة react native.
